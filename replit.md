# replit.md

## Overview
This is a Discord economy bot built with Python using Hikari, Lightbulb, and Miru. It features a coin reward system, daily rewards, streak multipliers, redemption workflows, and admin controls.

## Project Structure
- `main.py`: Entry point and bot initialization.
- `data.json`: Persistent storage for guild and user data.
- `bot.log`: Application logs.
- `extensions/`: (Planned) Modular command and listener files.

## Recent Changes
